search-fish
===========
## 概要
ダイビングスポットからそこに生息する魚を検索できるWEBアプリ

## 主な機能
ダイビングスポットを検索できる  
そのスポットで見られる魚の一覧を表示する  
googleの画像検索で一番最初の魚の画像を表示する
